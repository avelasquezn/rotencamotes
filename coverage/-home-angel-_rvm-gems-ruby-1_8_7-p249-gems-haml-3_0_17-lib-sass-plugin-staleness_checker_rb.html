5     # To speed things up two level of caches are employed:
 
6     #
 
7     # * A class-level dependency cache which stores @import paths for each file.
 
8     #   This is a long-lived cache that is reused by every StalenessChecker instance.
 
9     # * Two short-lived instance-level caches, one for file mtimes

11     #   These are only used by a single StalenessChecker instance.

13     # Usage:
 
14     #
